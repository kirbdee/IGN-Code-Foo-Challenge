LIQUID LAYOUTS

Create a liquid layout with HTML, CSS, and Javascript. This layout must support the following resolutions: 1024x768, 1680x1050. Explain why you would use a liquid layout.

This site uses a liquid/responsive/adaptive layout that supports 1680x1050, 1024x768, and mobile screen sizes.

Please feel free to resize the window's width or change your computer's resolution.

Using a layout that is responsive to different screen sizes allows users to experience the site or application in the most optimal way. For this site I simply used media queries and applied css to varying ranges of widths.
